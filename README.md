# 📚 Projeto_Portal_de_Conhecimento
*Base de Conhecimento*

# Objetivo
Sistema desenvolvido com o objetivo de centralizar soluções técnicas, tutoriais e respostas para dúvidas frequentes, reduzindo a carga de trabalho do suporte de TI.

# 📝 Descrição do Projeto
O Portal de Conhecimento é uma solução desenvolvida para com o intuito de mitigar a perda de tempo da equipe de TI com dúvidas repetitivas. O sistema permite que administradores postem artigos e tutoriais, enquanto usuários podem pesquisar soluções e fornecer feedback sobre a eficácia do conteúdo.

# 🛠️ Tecnologias Utilizadas: #
> Linguagem: Python;
> Framework: Django (Padrão MVT);
> Banco de Dados: SQLite;
> Interface: HTML5/CSS3 (Django Templates);
> Versionamento: Git & GitHub.
